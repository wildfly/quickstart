EJB in EAR throws custom exception Quickstart

EJB throws custom exception Quickstart

Clicking the 'Say Hello' button will call the EJB sending it the value of the 'Name' input text box and the EJB response will be displayed in the 'Response' output text
Clicking the 'Say Hello' button with an empty 'Name' box will send an empty string to the EJB resulting in the EJB throwing 'GreeterException' back, which the web app will catch and display the exception's message in the 'Response' output text
